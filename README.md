# SlimeGameProject

## Function

- GetInput() : 키보드, 마우스 등 입력을 받는 함수
- LookAround() : 마우스 이동 시 화면 전환 함수
- Move() : 이동 함수
- Run() : 달리기 함수
- Jump() : 점프 함수
- Attack() : 몸통 박치기 함수
- IEnumerator WaitforAttack() : 몸통 박치기에 딜레이를 걸어주는 코루틴 함수
- Interaction() : 원소를 획득하고 조건에 맞춰 공격력을 증가 시키는 함수
- Heal() : 체력을 회복 시키는 함수
